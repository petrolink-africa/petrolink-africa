# PetroLink Africa

This folder contains the launch assets for PetroLink Africa — a fully automated B2B petroleum trading platform.

## How to Use

1. Upload all files to GitHub or Google Drive
2. Share the live link with investors and freelancers
3. Use the QR code to onboard via WhatsApp
